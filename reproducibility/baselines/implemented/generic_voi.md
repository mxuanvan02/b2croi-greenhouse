# Generic VoI

## Status

Self-contained implementation in the shared B2CRoI-H(Q) simulator; no third-party official code is vendored.

## Selection rule summary

Ranks loops by a generic value-of-information score combining estimation mismatch and age.

## Role in evaluation

VoI-inspired baseline without the proposed hybrid safety-fairness mechanism.
