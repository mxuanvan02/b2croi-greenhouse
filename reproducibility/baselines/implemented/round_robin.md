# Round-robin

## Status

Self-contained implementation in the shared B2CRoI-H(Q) simulator; no third-party official code is vendored.

## Selection rule summary

Selects loops cyclically, independent of prediction error, channel state, and safety risk.

## Role in evaluation

Classical low-complexity scheduling baseline.
