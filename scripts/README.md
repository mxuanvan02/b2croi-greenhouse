# Scripts directory

Scripts are organized as reproducibility utilities for simulations, validation, figure generation, and table generation.

Suggested order:
1. Benchmark/validation scripts (`b2croi_*.py`, `second_dataset_validate_mendeley.py`).
2. Public table generation (`make_public_tables.py`, `csv_to_latex_tables_compact.py`).
3. Figure generation (`make_*figures*.py`).
